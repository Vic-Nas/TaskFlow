LCLICK
Performs a left mouse click at specified coordinates
Syntax:
LCLICK X,Y times description
Example:
LCLICK 500,300 1 Click start button
RCLICK
Performs a right mouse click at specified coordinates
Syntax:
RCLICK X,Y times description
Example:
RCLICK 200,150 1 Right-click context menu
KEY
Sends keyboard shortcuts and hotkey combinations
Syntax:
KEY key1,key2,key3 times description
Examples:
KEY ctrl,c 1 Copy text

KEY win,d 1 Show desktop

KEY alt,tab 1 Switch window
TYPE
Types text as if typed on keyboard
Syntax:
TYPE text times description
Examples:
TYPE Hello[SPACE]World 1 Type greeting

TYPE username@email.com 1 Enter email
WAIT
Pauses execution for specified seconds
Syntax:
WAIT seconds times description
Examples:
WAIT 2.5 1 Wait for loading

WAIT 0.1 1 Brief pause
OPEN
Opens files, folders, or applications
Syntax:
OPEN filepath times description
Examples:
OPEN C:\document.txt 1 Open document

OPEN notepad.exe 1 Launch Notepad
EXEC
Executes another TaskFlow task group file
Syntax:
EXEC taskfile.task times description
Example:
EXEC login_sequence.task 1 Run login tasks

Note: Use EXEC to chain task groups and bypass the 14-task limit

üîß Usage Workflow

Creating Your First Automation

1. Create New Group: Click "New Group" and name your task sequence
2. Detect Coordinates: Use "Detect Coords" to find exact click positions
3. Build Tasks: Add commands like LCLICK, TYPE, KEY in sequence
4. Test Individual Tasks: Use "Run" button on each task to verify
5. Run Full Group: Execute complete automation with "Run Group"
6. Share & Export: Save your task file and share with others
